## Integraciones se APAGAN si:

### 1) Complejidad
Si tomar datos de wearables retrasa el MVP:
→ apagar integraciones 30 días

### 2) Riesgo legal
Si no se logra consentimiento claro:
→ apagar integraciones

### 3) Datos inútiles
Si en 14 días los datos no generan valor percibido:
→ apagar integraciones

### 4) Dependencia externa
Si el proveedor bloquea acceso:
→ fallback manual permanente
