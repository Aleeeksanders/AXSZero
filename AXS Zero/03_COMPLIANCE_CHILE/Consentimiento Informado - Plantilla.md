## Consentimiento General (obligatorio)

Antes de usar Eris, declaro que:

1. Entiendo que **Eris no es un servicio médico ni terapéutico**.
2. Entiendo que los datos que ingreso son **autodeclarados**.
3. Autorizo a AXS a **procesar esta información solo para operar el servicio**.
4. Puedo solicitar **acceso, rectificación o eliminación** de mis datos.
5. Puedo dejar de usar el servicio en cualquier momento.

---

## Forma de Aceptación
- Checkbox explícito
- Fecha y versión del texto
- Registro local

El servicio **no inicia** sin esta aceptación.

---

## Consentimiento Adicional (opcional) — Wearables

Este bloque **solo se presenta** si el usuario decide conectar dispositivos.

> Acepto que Eris procese **datos fisiológicos agregados**
> (sueño, HR promedio, estrés si existe)
> provenientes de mis dispositivos conectados.

Entiendo que:
- Estos datos **no constituyen diagnóstico médico**
- Se usan solo para **dar contexto diario**
- Puedo **desactivar esta función cuando quiera**
- La experiencia base de Eris **no depende** de estos datos

Definición técnica en:
→ [[Integraciones_Wearables/Integración Wearables - Alcance]]

Sin consentimiento general → Eris no funciona
