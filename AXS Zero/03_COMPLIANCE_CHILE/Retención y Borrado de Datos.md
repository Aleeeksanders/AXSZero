# Retención y Borrado de Datos — Zero Phase

Relacionado con:

## Retención
- Datos activos: mientras el usuario use Eris
- Inactividad > 90 días → sugerir borrado

---

## Borrado
- A solicitud del usuario
- Eliminación lógica y física
- Confirmación explícita
