# Eris.Memory — Esquema mínimo (v0)

Relacionado con:
- [[Eris v0 - Alcance]]
- [[Modelo de Datos v0 (SQLite)]]

---

## Qué se guarda
- Fecha de sesión
- Texto del usuario
- Resumen generado
- Foco elegido

---

## Qué NO se guarda
- Inferencias emocionales profundas
- Diagnósticos
- Datos de terceros

## Regla
