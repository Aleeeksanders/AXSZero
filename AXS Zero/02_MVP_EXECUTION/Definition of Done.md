# Definition of Done — MVP

Relacionado con:
- [[MVP Scope v0]]
- [[Sprint Actual]]
- [[Demo Script]]

---

## Una tarea está DONE solo si:
- Funciona sin intervención tuya
- No requiere explicación técnica
- Aporta valor directo al usuario
- No rompe compliance básico

---

## El MVP está DONE solo si:
- Un usuario lo usa solo
- Da feedback claro
- No viola [[Mapa Legal (qué aplica y por qué)]]

---

## No es DONE si:
- “Casi funciona”
- “Después lo arreglamos”
- “Se entiende si explico”

---

## Regla Final
> DONE = usable por alguien que no eres tú.
