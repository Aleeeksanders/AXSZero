# Demo Script — Eris v0

Relacionado con:
- [[MVP Scope v0]]
- [[Propuesta de Valor (7 días)]]
- [[Experimento 01 - Hipótesis]]

---

## Objetivo
Probar valor, no impresionar.

---

## Guión Básico
1. Usuario abre Eris
2. Eris pregunta estado actual
3. Usuario escribe libremente
4. Eris ordena y refleja
5. Usuario responde
6. Eris sugiere un foco
7. Usuario cierra sesión

---

## Duración
- 3 a 7 minutos

---

> “¿Esto te ayudó a ordenar tu cabeza?”
