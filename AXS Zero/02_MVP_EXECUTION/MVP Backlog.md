## Backlog Prioritario (orden fijo)

### P0 — Obligatorio
- [ ] Flujo de conversación inicial
- [ ] Registro manual de estado mental
- [ ] Resumen diario simple
- [ ] Persistencia local de datos
- [ ] Consentimiento explícito (link)

### P1 — Solo si P0 está estable
- [ ] Etiquetas de contexto
- [ ] Historial navegable
- [ ] Feedback al cierre de sesión

### P2 — Congelado
- Automatismos
- Predicciones
- Visualizaciones

## Regla
Nada entra al backlog sin:
- vínculo a valor
- impacto en uso
- claridad de ejecución
